Um pangrama é uma frase que inclui todas as letras do alfabeto pelo menos uma vez. Para este problema, consideraremos uma versão simplificada do alfabeto português, excluindo as letras k, w, y e ignorando acentos. Sua tarefa é determinar se uma dada frase é um pangrama, levando em conta apenas letras minúsculas, espaços, vírgulas e dois pontos.

Entrada Uma string C, representando a frase a ser analisada.

Saída: Uma string contendo um único caractere: 'S' se a frase for um pangrama, ou 'N' caso contrário.
